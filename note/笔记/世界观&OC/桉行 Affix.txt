## 关于'' Key ''

某次他爬树时，看见天上掉下来了一个东西，走近一看发现是块金属，桉行觉得这石头漂亮，又是从天上掉下来的，就把它带回去研究，最终做成了 Key

Key 看起来就像一颗十字星，平时缀在桉行的尾巴尖上，很轻很坚硬。由桉行使用天外陨石制成  `(后续得知天外陨石是星辰钢)`  平时就像一个普通的装饰品，总是和桉行形影不离。

经过桉行的改造，Key 已经成为了一个多功能工具，拥有各种各样神奇的功能。桉行把自家门禁卡塞了进去，回家时尾巴一扫就能开门...这大概也是桉行管它叫 Key 的原因。  

同时，桉行也把自己的手机系统装了进去，也就是说，现在 Key 也相当于是桉行的手机... `(幸好学校不知道，桉行也不会怎么在学校用 Key 玩手机。)`

在战斗中，桉行通常会将 Key 甩出去，操纵 Key ，用尾巴指挥 Key 进行战斗 `(通常尾巴摆动幅度很轻，几乎看不出来，若尾巴受伤会较大的影响 Key 的精度)` 即使桉行停止施加魔法， Key 也能在空中滞留一段时间。`(使用 Key 本身并不需要魔法)`

Key 能结合桉行的空间魔法，能够变成几乎任何桉行能想到的工具，也用魔法增加 Key 的重量。在战斗中，桉行通常会增加 Key 一定重量并给予 Key 一个锋利的边缘。

## 关于魔法 Magic

由于某些原因，桉行并不想让别人知道他会魔法，但他确实是 Magus ，所以他一直隐藏着，只有在附近没有人时或是危急关头才使用。桉行会星辰，空间，天域三种魔法 `(其实主要还是星辰魔法，其他的他学的并不好)`

---

## 星辰

桉行的星辰魔法不是与生俱来的。而是来源于父亲书房里的魔法书《星予（Star-Blessed）》  

这本书通俗易懂，直接讲的就是如何和星星建立联系，如何使用自己的意念力之类的，几乎没有一个多余的字。虽然书里没有教具体的招式，但桉行还是就借着这本书，学习了他第一个魔法——星辰魔法  

桉行能够借助星辰魔法，召唤一些魔法星星，或是直接施加在 Key 上让 Key 发光点亮周围等等，~~具体招式我在这里就不细讲了反正很厉害~~  

---

## 空间  

空间魔法来自于桉行的父亲。有时，他的父亲会去开发区练习魔法，桉行则会躲在旁边偷看。虽然他的父亲并没有教他，但他也默许了桉行的偷师。

桉行能借用空间魔法，制作一些临时形成的工具，或是用来加工一些正常难以加工的物品 `(比如星辰钢)` 同时，他也能用空间魔法进行短距离的空间折跃，不过据他所说挺消耗体力。

---

## 天域

来源于某日桉行在网上购买的新鲜玩意，是一根来自天域的魔法羽毛。想要使用这个羽毛写字必须掌握一定天域魔法，幸好说明书上有写一些基础的内容，桉行借助说明书和对天域羽毛的研究学习了一定的天域魔法。` (但是大部分时候，桉行使用天域魔法都必须借助天域羽毛) `

## Key

桉行不会随意使用魔法，面对一些简单的威胁时 `(例如他的同学)` 用 Key 解决他们就够了

桉行将 Key 从尾巴上甩出并用尾巴指挥 Key 攻击，使用 Key 无需魔法，它能够利用自身能源在空中滞留很久，必要时返回桉行也很快

Key 本身是星辰钢材质，质量较轻；但它是尖锐的四角星形状。用它来对付同学不容易让同学受伤但又能让同学感到痛，就算遇到特殊情况，稍微向 Key 加入一些魔法也不会被发现，用来教训同学真是再适合不过了  

通常在桉行甩出Key后，他只需要在旁边坐着轻轻摇摇尾巴或是走位一下就能打败同学了，即使同学能够近身，身手敏捷的桉行也能灵巧的躲过，~~顺带咬他一口~~， Key 用来对付两三个 Magus 同学都是绰绰有余的  

## Magic

桉行能够使用星辰魔法，快速地在身边召唤一些星星并将它们射向目标，也能够在目标周围召唤一些星星连成星座。攻击来自四面八方且速度很快

```
(虽然桉行能召唤这些星座，但其实桉行并不认识它们)
```  

Key将会受到来自桉行的魔法强化，在空中时质量增加并会拥有范围更大更锋利的十字魔法强化边缘，此时桉行并不会将它长久地留在空中，而是使用一次后就收回 `(就像回旋镖一样)` 既是为了节省体力，也是为了重新蓄力再次攻击 `(桉行也能够直接甩动尾巴用尾巴上的Key攻击)` 

桉行也能够利用空间魔法进行短距离的折跃来躲避攻击，但据他所说较为消耗体力不能用的过于频繁。桉行也能利用头上的天域羽毛召唤一些魔法羽毛，相较于星星，旋转的羽毛的运动轨迹可以是曲线

# 一些其他的闲话...  

---

桉行平时很珍惜时间，能用跑的绝对不用走的，就算是走路也是小跑一样的速度。讨厌别人挡他的道。楼梯间偶遇牵手姐时，他会趁没有人注意，试着传送到前面去。上课赶不及时，会尝试用最消耗体力的向上传送。  
`(顺带一提，桉行的传送范围大约是10米，但想传送别人或物品，则必须靠的桉行很近或者被桉行碰到)`

---

桉行并不好战，但他又喜欢表现自己。在战斗中若你直接向他求饶认输，或是夸他几句好听的，他大概就直接放过你了

当然，要是他感到情况不对他打不过你时，可能会找机会逃跑，实在无法逃脱时，卖萌撒娇求饶认输什么的也不是不行...
